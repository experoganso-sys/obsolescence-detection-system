# obsolescence-detection-system
Prototype de système automatisé de détection de l'obsolescence des logiciels dans un parc informatique

# Système de détection de l'obsolescence des logiciels

Ce projet a été réalisé dans le cadre d’un mémoire de licence en sécurité informatique.

## Objectif

Développer un prototype capable d’analyser un inventaire logiciel et de détecter les logiciels obsolètes dans un parc informatique.

## Technologies utilisées

- Python
- Fichier CSV pour l'inventaire

## Exécution

Pour exécuter le script :

python obsolescence_checker.py     -inventaire.csv
